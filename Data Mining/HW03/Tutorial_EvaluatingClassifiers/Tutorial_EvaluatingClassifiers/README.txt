This archive contains a Jupyter Notebook. To use it, you need to have installed:

1) Python and the following libraries: numpy, scipy, pandas, matplotlib, and scikit-learn
2) IPython - https://ipython.org/install.html
3) Jupyter - http://jupyter.readthedocs.io/en/latest/install.html


Opening a Jupyter/IPython Notebook
----------------------------------

If you have all the required software installed, open the terminal and type (substitute the correct ipynb file):

$ jupyter notebook downloaded_exercise_notebook.ipynb
